# Description

Write a recursive function called `Intersect`. Given two arrays, write a function to compute their intersection.

# Example Output

|              Input              | Output |
| :-----------------------------: | :----: |
|   Intersect([1,2,2,1], [2,2])   | [2,2]  |
|                                 |        |
| Intersect([4,9,5], [9,4,9,8,4]) | [4,9]  |
